## ğŸ“ Overview

**Trendify** is a modern, full-stack e-commerce application built with **Next.js (App Router)**. It provides a smooth shopping experience from secure authentication to order management, featuring a clean and responsive UI designed for fashion enthusiasts.

---

## âœ¨ Key Features

* **Authentication:** Full Login and Register system with secure validation.
* **Product Catalog:** High-quality image displays for categories like Fashion, Electronics, and more.
* **Shopping Cart & Wishlist:** Seamlessly add, remove, and manage your favorite items.
* **Order Management:** Track and view all your orders in a dedicated table.
* **Responsive UI:** Optimized for a perfect experience on mobile and desktop using Tailwind CSS.
* **404 Custom Page:** A user-friendly error page for a better UX.

To run this project locally on your machine, follow these steps:

### 1. Clone the repository
Open your terminal (Git Bash or VS Code terminal) and run:
